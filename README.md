## Interview Prep

A concise, structured collection of guides to prepare for DevOps and other interviews.

---

## Overview

This repo organizes topics like DevOps fundamentals, Docker, Kubernetes, and Python into focused Markdown guides.
Each guide covers core concepts, common interview questions, and practical, hands-on tasks.

---

## How to use

- Start from the roadmap in devops/.
- Deep dive into topic files (docker.md, kubernetes.md, python.md, etc.).
- Use interview-sets/ to simulate real interview rounds.

---

## Bonus

This is managed using Obsidian, an open source note taking app. You can download it from - https://obsidian.md/

### Some common settings for obsidian

- Obsidian > Settings > Default Location for new attachemnts > In Subfolder Under Current Folder
- Obsidian > Settings > Default Location for new attachemnts > Subfolder Name - attachments
- Obsidian > Settings > Default Location for new attachemnts > New Link Format - Relative path to file
- Obsidian > Settings > Default Location for new attachemnts > Use `[[Wikilinks]]` - Disable It
